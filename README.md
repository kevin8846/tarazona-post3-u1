#  Manejo de DEBUG en DOSBox — Unidad 3

Laboratorio desarrollado para la materia **Arquitectura de Computadores** utilizando **DOSBox y DEBUG** para explorar registros, memoria y desensamblado en modo real.

El objetivo principal fue aprender el uso de comandos básicos del depurador DEBUG como `R`, `D`, `F`, `U` y `A`, además de comprender cómo se representan los datos y las instrucciones en memoria.

---

#  Objetivos del Laboratorio

Configurar el entorno DOSBox  
Montar una unidad virtual  
Utilizar el depurador DEBUG  
Inspeccionar registros del procesador  
Rellenar y visualizar memoria  
Explorar el PSP  
Desensamblar instrucciones  
Ensamblar un pequeño programa en memoria  

---

#  Tecnologías y Herramientas

- DOSBox 0.74-3
- DEBUG
- Git & GitHub
- Visual Studio Code
- Windows / Linux / macOS
